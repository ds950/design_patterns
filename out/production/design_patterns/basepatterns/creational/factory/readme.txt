Паттерн: Фабрика (FactoryMethod)
-------------------------------------------------------
Цель:

Создание интерфейса, который создает объект. При этом,
выбор того, экземпляр какого класса создавать остается
за классами, которые имплементируют данный интерфейс.
-------------------------------------------------------
Зачем:

Для делигирования создания экземпляров другому классу.
-------------------------------------------------------
Пример:

- заранее неизвестно, экземпляры какого класса нужно
будет создавать;
- класс спроектирован таким образом, что создаваемые
им объекты имеют свойства определенного класса.
-------------------------------------------------------
